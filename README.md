# Excel Data Analysis Projects

This repository contains beginner-level data analysis projects completed using Microsoft Excel.
The projects focus on practicing foundational skills such as data cleaning, data exploration,
basic analysis, and simple data visualization using structured datasets.

## Projects

### 1. Bike Sales Analysis
Exploratory analysis of a bike sales dataset using Excel. The project focuses on organizing data,
using pivot tables, and creating simple visualizations to understand customer demographics
and purchasing patterns.

### 2. World Famous Places Analysis
Basic analysis of a dataset containing world-famous places. This project focuses on data cleaning,
sorting, filtering, and summarizing information by location and category using Excel.

## Tools Used
- Microsoft Excel
